TP final Node.js

Bienvenue dans le TP final de Node.js. Ce TP consiste à créer une API RESTful générique capable de gérer différentes entités.

Instructions

  1. Cloner le dépôt GitHub fourni.
  2. Installer les dépendances avec npm install.
  3. Configurer la base de données MongoDB.
  4. Lancer le serveur avec npm start ou en mode développement avec npm run dev.
  5. Tester les différentes routes de l'API pour gérer les entités.
